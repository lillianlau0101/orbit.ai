orbit.ai | 你的情绪卫星 🛰️

orbit.ai 不仅仅是一个日记本，它是你情感世界的观察者。通过与 AI 伴侣“零零”的日常对话，它能自动捕捉你生活中的细微波动，并将其转化为可视化的洞察。

🌟 核心打磨点 (Product Highlights)
温柔共情 (Empathic AI)：基于混元 Pro 大模型，提供不超过 75 字的温情反馈，拒绝冷冰冰的机器回答。

情绪建模 (Mood Quantification)：自动将文字转化为 1-10 的情绪分值，帮助你复盘心情曲线。

自动洞察 (Smart Insight)：无需手动打标签，AI 会自动从对话中提炼出“事件”、“决策”与“关联点”。

米色极简 (Zen UI)：采用高质感的米色系设计，营造安全、静谧的倾诉空间。

🛠️ 技术架构 (Tech Stack)
Frontend: 微信小程序原生框架 (Base Lib 3.x)

Backend: 微信云开发 (WeChat CloudBase)

AI Engine: 腾讯混元 Hunyuan-pro (Cloud Integration)
