orbit.ai 🛰️ | 你的心理健康卫星
orbit.ai 是一款基于腾讯混元大模型 (Hunyuan-pro) 的治愈系 AI 情绪日记。它像一颗卫星一样环绕着你的日常生活，通过对话捕捉情绪，并将其转化为可视化的成长洞察。

✨ 产品亮点
零距离陪伴：AI 伴侣“零零”提供极具共情力的温情对话。

情绪量化：自动分析对话内容，量化为 1-10 的情绪分值。

多维洞察卡片：自动提炼核心事件、情绪触发点及生活决策。

米色极简主义：专为减压设计的视觉系统，营造安全倾诉空间。

🛠️ 技术栈
Frontend: 微信小程序原生框架 (Base Lib 3.x)

Backend: 微信云开发 (WeChat CloudBase)

AI Model: 腾讯混元 Hunyuan-pro

🚀 快速开始
克隆本项目到本地。

打开 app.js，将 env 字段替换为你自己的微信云开发环境 ID。

在微信开发者工具中右键点击 cloudfunctions 下的函数，选择“上传并部署”。
